# Hans Restaurant App

Welcome to the Hans Restaurant App documentation. This is a modular restaurant management application built with modern web technologies.

## Quick Start

- [Getting Started](/getting-started) - Set up your development environment
- [Project Tooling](/tooling) - Development tools and workflows
- [Architecture](/architecture) - System design and structure
- [API Reference](/api) - API documentation

## Features

- **Modular Design**: Scalable architecture for restaurant operations
- **Real-time Updates**: Live data synchronization across devices
- **Multi-platform**: Web and mobile support
- **Analytics**: Comprehensive reporting and insights

## Tech Stack

- **Frontend**: Next.js 15, React, TypeScript
- **Backend**: Supabase, Edge Functions
- **Database**: PostgreSQL
- **Hosting**: Vercel
- **Monitoring**: Sentry, PostHog
