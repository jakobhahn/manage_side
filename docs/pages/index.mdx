# Hans Restaurant Management Platform

Welcome to the Hans Restaurant Management Platform - a comprehensive, modular SaaS solution for restaurant management with integrated payment processing and mobile-first design.

## Platform Overview

The Hans Restaurant Management Platform is a cutting-edge SaaS solution that provides restaurants with modular management tools. Customers can subscribe to individual modules based on their needs, with a unified payment system integration (SumUp) as the central data source.

## Technology Stack

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **Payment**: SumUp API with n8n automation
- **Mobile**: PWA with responsive design
- **Development**: Cursor AI, Turborepo, pnpm
- **Deployment**: Vercel with GitHub Actions CI/CD
